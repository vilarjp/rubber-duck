# Context And Direction

Use this reference before major design or redesign work. The goal is to replace category reflexes with a product-specific point of view.

## Frame The Human

Answer these before committing to visual choices:

- **Human**: Who is using this? Name the role, skill level, emotional state, and environment.
- **Job**: What are they trying to accomplish in one verb phrase?
- **Scene**: Where and when is this opened? Bright office, phone in transit, midnight incident room, sales call, kitchen counter, showroom?
- **Pressure**: What makes the task hard: time, risk, uncertainty, volume, trust, boredom, compliance, coordination?
- **Success**: What should the person understand, decide, or do faster because of this UI?

If the answers are unknown and design identity depends on them, ask. If the choice is reversible, infer and keep moving.

## Choose The Register

- **Product**: The interface is a tool. Users return often, compare information, complete workflows, and need trust. Favor clarity, consistency, dense-but-readable structure, standard affordances, restrained motion, and reusable components.
- **Brand**: The interface is part of the message. Visitors form an impression. Favor stronger point of view, imagery or art direction, distinctive type, memorable composition, and more expressive motion.
- **Hybrid**: A public product surface with working UI, a SaaS marketing page with live demos, onboarding, docs, pricing, or a logged-out/logged-in bridge. Use brand energy at entry points and product discipline once tasks begin.

Do not let the URL or category decide alone. A dashboard can be ceremonial for executives. A landing page can be utilitarian for developers.

## Direction Inventory

Create a concise internal inventory:

- **Domain cues**: Materials, tools, places, rituals, data shapes, vocabulary, and real-world objects that belong to this product.
- **Color world**: Colors that come from the product's world, not from generic category expectations.
- **Structure metaphor**: Workspace, instrument panel, ledger, studio, archive, map, timeline, command center, catalog, notebook, gallery, canvas, marketplace, terminal, table, checklist, queue.
- **Signature**: One visual, structural, or interaction idea that could only belong here.
- **Defaults to reject**: Three obvious patterns for this category and what replaces them.

The direction passes when removing the product name still leaves clues about what the product is for.

## Style Axes

Set these as deliberate values:

- **Density**: compact, comfortable, spacious, cinematic.
- **Expression**: invisible utility, refined product, memorable product, expressive brand, experimental.
- **Softness**: sharp, tailored, friendly, pillowy.
- **Depth**: flat-border, tinted surfaces, soft shadows, dimensional, immersive.
- **Temperature**: cool, neutral, warm, mixed, high-contrast.
- **Motion**: quiet feedback, fluid continuity, choreographed reveal, playful delight, reduced.
- **Trust posture**: clinical, editorial, handcrafted, premium, technical, energetic, calm.

These axes become tokens and layout rules, not adjectives in a note.

## Product Guidance

For dashboards, admin panels, SaaS apps, tools, and authenticated surfaces:

- Familiar affordances are strengths when they reduce cognitive load.
- Use consistency as an interaction feature: same action hierarchy, same table behavior, same form flow, same empty-state vocabulary.
- Save distinctiveness for structure, information hierarchy, domain-specific details, and a few memorable moments.
- Use restrained palettes unless the product's purpose demands stronger color.
- Prefer stable type scales, durable spacing, and predictable navigation.
- Avoid decorative animation, display fonts in dense controls, reinvented form fields, and over-styled buttons that make common actions feel risky.

## Brand Guidance

For landing pages, portfolios, launches, editorial surfaces, product stories, and campaigns:

- A strong point of view is required. Average is invisible.
- Imagery, typography, composition, and pacing carry the product's promise.
- Commit to a color strategy. Restrained can work, but only when it is precise and intentional.
- Avoid generic centered hero stacks, repeated feature cards, decorative icon rows, and "tech purple gradient" reflexes.
- When the subject is physical, visual, spatial, culinary, fashion, travel, real estate, art, wellness, or people-centered, ship real imagery or generated bitmap assets. Empty colored boxes are not art direction.
